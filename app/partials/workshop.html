Introduction

a 7*7 Gomoku board is used here. Empty board is represented by the string '_______,_______,_______,_______,_______,_______,_______', every part separated by the comma represents one row of the board.

The Gomoku bot that you can create is basically a function that determines its next move based on current board. That is, the function will return new board based on previous board (Parameter #1, with default name 'board') and your player mark ('X' or 'O')(Parameter #2, with default name 'side').

Pick an interesting name and write your bot, you can either start from an existing bot or write a fresh one. Once finished, you can use 'Test JavaScript/Python code' button to test your bot. After testing, click 'Add bot' button to save it permanently.

Note: For now, Javascript and Python is supported, more language options will be added soon. Feel free to create your first Gomoku bot now!
