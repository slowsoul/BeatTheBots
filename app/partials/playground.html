Introduction

a 7*7 Gomoku board is used here. Empty board is represented by the string '_______,_______,_______,_______,_______,_______,_______', every part separated by the comma represents one row of the board.

The Gomoku bot that you can create is basically a function that determines its next move based on current board. That is, the function will return new board based on previous board and your player mark ('X' or 'O'). Three example bots written in JavaScript and Python are provided below.

Once you have selected two bots, you can make them play and watch the steps and result by hitting the Play Game button.

Note: For now, Javascript and Python is supported, more language options will be added soon. Feel free to create your first Gomoku bot now!
